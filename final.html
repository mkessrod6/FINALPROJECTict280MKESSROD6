Project Overview

📋 Project Description

This website was built for GreenTech Solutions, a fictional environmental sustainability and technology startup. The site serves two primary audiences: small to medium-sized businesses (SMBs) seeking eco-friendly technology solutions, and environmentally conscious individuals interested in sustainable tech.

The site contains five full content pages (Homepage, About Us, Products, Blog, Contact) plus this documentation page, a shared CSS stylesheet, and a JavaScript file providing interactivity across all pages.

🛠️ Technologies Used

  • ✅ HTML5 — Semantic structure with accessibility attributes
  • ✅ CSS3 — External stylesheet, custom properties, media queries
  • ✅ JavaScript (ES6+) — DOM manipulation, form validation, IntersectionObserver, animations
  • ✅ Google Fonts — Playfair Display + DM Sans
  • ✅ Responsive Design — Flexbox, CSS Grid, mobile-first breakpoints
  • ✅ WCAG 2.1 — ARIA roles, alt text, keyboard navigation, color contrast

✅ Rubric Requirements Checklist

Theme & Purpose (20 pts)

  • 🟢 Creative eco-tech theme aligned with GreenTech client profile
  • 🟢 Clear purpose: educate, engage, convert sustainability clients
  • 🟢 Target audience: SMBs + eco-conscious individuals
  • 🟢 Functioning navigation bar on every page

Semantic HTML5 (20 pts)

  • 🟢 <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>
  • 🟢 Headings (h1–h4), paragraphs, lists (ul/ol)
  • 🟢 Forms with fieldsets, labels, inputs, textarea, select
  • 🟢 Tables, <time>, <address>, <details>/<summary>
  • 🟢 5+ distinct element types used per page

CSS Styling (20 pts)

  • 🟢 External stylesheet (css/styles.css)
  • 🟢 CSS custom properties (variables) throughout
  • 🟢 Flexbox and CSS Grid layouts
  • 🟢 CSS animations (@keyframes, transitions)
  • 🟢 Hover/focus states on all interactive elements

JavaScript (20 pts)

  • 🟢 Contact form validation with error messages
  • 🟢 Product search + category filter (dynamic content)
  • 🟢 Scroll-reveal animations (IntersectionObserver)
  • 🟢 Animated number counters
  • 🟢 Newsletter success feedback
  • 🟢 Mobile hamburger menu toggle
  • 🟢 Accessibility toolbar (font size, contrast)

Mobile Responsiveness (20 pts)

  • 🟢 Responsive grid layouts (auto-fit/minmax)
  • 🟢 Media queries at 1024px, 768px, 480px
  • 🟢 Mobile hamburger nav replacing desktop menu
  • 🟢 Fluid typography with clamp()
  • 🟢 Tested at 320px, 375px, 768px, 1024px, 1440px

Web Accessibility (20 pts)

  • 🟢 Skip-to-content link on every page
  • 🟢 ARIA roles, labels, aria-live regions
  • 🟢 All images have descriptive alt text
  • 🟢 Sufficient color contrast (WCAG AA+)
  • 🟢 Keyboard-navigable forms and controls
  • 🟢 High-contrast mode toggle in accessibility toolbar
  • 🟢 Font-size adjustment controls

🚀 Setup & Running the Project

  1. Upload the entire greentech/ folder to your NMSU web server at ~/public_html/
  2. Ensure the file structure is preserved exactly (css/, js/, pages/ subdirectories).
  3. Access the site at https://web.nmsu.edu/~YOURUSERNAME/greentech/
  4. No build tools, npm, or server-side code required — this is a pure static site.
  5. All fonts load from Google Fonts CDN (requires internet connection).

Local testing: Open index.html directly in a browser, or use VS Code Live Server for accurate relative-path resolution.
